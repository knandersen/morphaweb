usage

1. drag one or more audiofiles onto the page

2. add markers using the shortcut listed below

3. export the wav-file to use with morphagene

SHORTCUTS

j
add marker
k
remove selected marker
␣
play/pause
this is an open source project by kevin nørby andersen. all audio data is processed client-side, which means it stays with you and only you. the app uses analytics for the sole purpose of tracking number of visits and errors.
